## Overview

This project implements an **Agentic AI system for Business Intelligence** using **Google's Agent Development Kit (ADK)**.

The system converts natural language questions into SQL queries, executes them on a database, and returns results with **visualizations and analytical explanations** generated by Google Gemini.

The repository contains:

- Agent implementations
- SQL guardrails and database connector
- Prompt templates
- Web interfaces (ADK Web and Gradio)
- Reproducible environment setup using **uv**

For detailed architecture and evaluation, see the **Short Technical Report**.

---

## System Pipeline

```
User Question
      ↓
Text-to-SQL Agent
      ↓
SQL Executor
      ↓
Business Intelligence Service
      ↓
Visualization + Explanation
```

The system follows a **sequential agent pipeline**, where each component performs a specific task in the data analysis workflow.

## Setup

### 1. Clone the repository

```bash
git clone https://github.com/ChronicleOfZealvy/Agentic_AI_Project.git
cd Agentic_AI_Project
cd gradio-adk-agent
```

### 2. Install dependencies

```bash
uv sync
```

This command installs all dependencies and creates the environment.

---

### 3. Configure Environment Variables

Create a `.env` file inside the `bi_agent` folder.

Example configuration:

```
GOOGLE_API_KEY=your_gemini_api_key_here

MSSQL_SERVER=your_server_address
MSSQL_DATABASE=your_database_name
MSSQL_USERNAME=your_username
MSSQL_PASSWORD=your_password
```

The `.env` file is ignored by Git to protect credentials.

---

## Running the Project

### Run with ADK Web

```bash
uv run adk web . --port 8000
```

Open:

```
http://127.0.0.1:8000
```

---

### Run with Gradio

```bash
uv run app.py
```

Open:

```
http://127.0.0.1:7860
```

Both interfaces use the same agent pipeline.
